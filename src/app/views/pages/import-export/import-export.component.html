üì¶ Products Import Format

Your CSV file must include the following columns:

  • sku- Unique product code (required)
  • name- Product name
  • brand_id- Related brand ID (get id from Brands table column name is ID)
  • category_id- Related category ID (get id from Categories table column name is ID)
  • unit_id- Related unit ID (get id from Units table column name is ID)
  • cost_price- Purchase price
  • sale_price- Selling price
  • stock- Available quantity
  • description- product description
Example: 
sku, name, category_id, unit_id, cost_price, sale_price, stock, description
  P-101, Pepsi, 2, 1, 60, 80, 50, Soft drink bottle
  P-102, Lays, 3, 1, 40, 60, 100, Potato chips

üì¶ Stocks Import Format

Used to import stock quantities or adjustments.

  • product_id- Related product ID (required)
  • quantity- Stock quantity (required)
  • warehouse_id- Warehouse or location (optional)
product_id,quantity,warehouse_id

üë§ Customers Import Format

Your CSV file should include these columns:

  • code- Unique customer code (required)
  • name- Full name
  • cnic- CNIC / National ID
  • email_address- Email address
  • mobile_number- Mobile phone number
  • phone_number- Landline (optional)
  • whatsapp- WhatsApp number
  • city_id- Related city ID
  • credit_balance- Current balance
  • credit_limit- Maximum credit limit
  • address- Full address
code, name, cnic, email_address, mobile_number, phone_number, whatsapp, city_id, credit_balance, credit_limit, address
  CT-001, Ali Khan, 12345-6789012-3, ali@gmail.com, 03123456789, , 03123456789, 12, 5000, 10000, Lahore Cantt
  CT-002, Ahmed Raza, 98765-4321098-7, ahmed@gmail.com, 03451234567, , 03451234567, 15, 2000, 5000, Karachi
